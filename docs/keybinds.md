|---|---|
| M | Telefoon |

| H | Fluiten |
| Shift+F2 | Porto GUI |
| Capslock | Praten in Porto|
| Rechter Alt |Fake porto praten|
| F1 | Wapen Attachments |
| F2 | Inventory |
| F3 | Emotes |
| F5 | Testserver Menu |
| F6 | Speler Interactie |
| F7 | Factuur Menu |
| F10 | Spelerlijst |
| F11 | Kleding |
| B | Wijzen|
| K | Accessoires menu |
| 3 | Emotes Menu|
| Shift + F9 (3sec) | Noodknop Politie |
## Auto Keybinds

| Toets | Werking |
|---|---|
| U | Auto Lock |
| L | Kofferbak |
| 3 | Car Menu |
